# Trading Bot


## Used MACD and MA strategy # Invest_IQ

## Need to have a env 

### Mac
```
python3 -m venv .venv
```

### Window
```
py -3 -m venv .venv
```

## Activate the environment 

### Mac
```
.venv/bin/activate
```

### Window 
```
.venv\Scripts\activate
```

## need to make a config.py file for your API keys

```python
BASE_URL= ''
API_KEY = ""
SECRET = ""
```

you will need to add your API keys in here
